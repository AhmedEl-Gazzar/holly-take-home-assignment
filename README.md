# Holly Engineering Take-Home Submission

## Set-up

- Install dependencies using `npm install`
- Set up environment variables by copying the `.env.example` file to `.env`
- Get a free API key from [Google AI Studio](https://aistudio.google.com/app/apikey) and add it to your `.env` file as `GEMINI_API_KEY`
- Run the application using `npm run dev`

## Usage

- Open `http://localhost:3000` in your browser and navigate to `/chat`
- Start typing your question in the chat input and press enter or click the send button
- The application will use the Google Gemini API to generate an answer to your question
- The answer will be displayed in the chat window
